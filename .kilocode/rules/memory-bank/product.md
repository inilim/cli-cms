# CLI-CMS Product Description

## Purpose
CLI-CMS - это минималистичная система управления содержимым (CMS), которая работает исключительно через командную строку (CLI) без использования веб-сервера. Проект предназначен для управления содержимым с помощью PHP-скриптов, запускаемых из командной строки. Система использует архитектурный подход, близкий к MVC, с внедрением зависимостей (DI) и шаблонизацией Twig для генерации HTML-контента в CLI-приложении. Поддерживает структурированный контент в формате Editor.js с обработкой различных типов блоков (заголовки, параграфы, списки, цитаты, код и др.) через сервис `BlockProcessingService` и шаблоны Twig. Использует модификатор `protected(set)` для свойств сущностей, позволяя контролировать доступ к данным.

## Core Functionality
- Управление содержимым через CLI-интерфейс
- Хранение данных в SQLite базе данных
- Использование архитектурного подхода, близкого к MVC с DI-контейнером
- Поддержка кроссплатформенности (разработка на Windows, эксплуатация на Ubuntu)
- Поддержка структурированного контента в формате JSON (например, Editor.js)
- Шаблонизация через Twig для генерации HTML-контента в CLI-приложении
- Обработка различных типов блоков контента (заголовки, параграфы, списки, цитаты, код и др.)
- Поддержка специфичных функций SQLite (CRC_32, UNIX_MS) для расширенной обработки данных
- Использование UUIDv7 в качестве идентификаторов записей

## User Experience
- Взаимодействие с системой происходит через командную строку
- Все функции системы запускаются через CLI-эндпоинты
- Система предоставляет инструменты для работы с данными через репозитории
- Встроенные механизмы логирования и обработки ошибок
- Поддержка структурированного контента с обработкой различных типов блоков (заголовки, параграфы, списки, цитаты и др.)
- Возможность кастомизации отображения через шаблоны Twig

## Problem Solving
- Предоставляет альтернативу традиционным веб-CMS для сценариев, где не требуется веб-интерфейс
- Упрощает разработку и развертывание CMS-подобных приложений без необходимости настройки веб-сервера
- Обеспечивает легковесное решение для управления структурированными данными
- Позволяет автоматизировать операции управления содержимым через скрипты
- Поддерживает современные форматы контента (например, Editor.js JSON)
- Обеспечивает гибкость в отображении контента через шаблонизацию Twig
- Предоставляет инструменты для безопасного доступа к данным через сущности с модификатором `protected(set)`
